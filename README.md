# Software FJ

## Descripción

Software FJ es una aplicación desarrollada en Python utilizando Programación Orientada a Objetos (POO) y Tkinter. El sistema permite gestionar clientes, servicios y reservas sin utilizar motores de bases de datos, almacenando la información únicamente en memoria y registrando eventos y errores en un archivo de logs.

## Características

- Registro de clientes.
- Gestión de servicios.
- Creación de reservas.
- Manejo de excepciones personalizadas.
- Registro de eventos y errores en un archivo de logs.
- Interfaz gráfica desarrollada con Tkinter.
- Simulación de operaciones válidas e inválidas.

## Tecnologías utilizadas

- Python 3
- Tkinter
- Programación Orientada a Objetos (POO)
- Git y GitHub

## Principios de Programación Orientada a Objetos implementados

- Abstracción
- Herencia
- Polimorfismo
- Encapsulación
